# Rosalie Law

This VitePress site is hosted from GitHub.io pages at [www.rosalielaw.com](https://www.rosalielaw.com).

## Setup / Requirements

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
nvm install
nvm use
npm install
```

### Development

```bash
npm run docs:dev
```

### Build

```bash
npm run docs:build
```

### Preview build

```bash
npm run docs:preview
```
